# Trash Walk - Server

## Tech Stack:

### Front-end:

### Back-end: 

- Koa

# Using Trash Walk - Server

A few things you have to take in consideration before using Trash Walk - Server

After cloning the repo you'll have to :

### Install global and local dependancies:

* NPM  `npm install`

* Node `brew install node`

## Start the server

In the shell type:

`$ npm start`
